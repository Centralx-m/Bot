# XTAAGC Bot - Production Trading Platform

## Features
- 🤖 REAL Bitget Integration
- 🔥 REAL Firebase Auth & Database
- 📊 Real-time Dashboard
- 💹 Live Trading Interface
- 🔒 Secure Authentication
- 🚀 Vercel Ready
- 🐳 Docker Support

## Quick Start
1. `pip install -r requirements.txt`
2. `cp .env.example .env` (add your REAL credentials)
3. `python backend.py`
4. Open `index.html`

## Deployment
- Vercel: `vercel --prod`
- Docker: `docker-compose up -d`
